    Форматирование текста

  • font-family: - стиль шрифта.
  • @font-face{} - подключение шрифтов из каталога сайта. Где src= отвечает за расположение, а font-family за используемое название. Иначе через link в шапке файла разметки
  • font-size: - размер шрифта.
  • font-weight: - жирность текста.
  • font-style: - стиль текста (italic).
  • text-align: - горизонтальное выравнивание контента.
  • text-align: justify; - регулирует расстояние между словами что бы текст занимал всю ширину блока.
  • text-decoration: - подчеркивание текста.
  • text-transform: - преобразование текста, например в нижний или верхний регистр.
  • border: - рамка (пар.: толщина, стиль, цвет).
  • cursor: - меняет значение курсора.
  • background-color: - меняет фоновый цвет елемента или блока.
  • white-space: - перенос текста.

    Форматирование блоков и елементов

  • text-align: - горизонтальное выравнивание элементов.
  • margin: 0 auto; - горизонтальное выравнивание элементов. по вертикали не работает.
  • border: - рамка (пар.: толщина, стиль, цвет).
  • border-width: - толщина рамки. Значение задаётся как в margin, padding
  • border-style: - стиль рамки.
  • border-color: - цвет рамки.
  • border-radius: - радиус углов рамки.

    Форматирование блоков и елементов

  • vertical-aligh: - вертикальное выравнивание элементов.
  • margin: 0 auto; - горизонтальное выравнивание элементов. по вертикали не работает.
  • border: - рамка (пар.: толщина, стиль, цвет).
  • border-width: - толщина рамки. Значение задаётся как в margin, padding
  • border-style: - стиль рамки.
  • border-color: - цвет рамки.
  • border-radius: - радиус углов рамки.

helper text elements

<br> - тег переноса текста на новую строку.

inline elements

стиль height, width к ним не применим(-ы)
занимают минимально необходимое место для контента.
Для строчных элементов нельзя задать цвет фона всей секции и установить её размеры.
Можно выравнивать по вертикали с помощью свойства vertical-align.
line-height меняет отступы между строками. не меняет высоту элемента.
вертикальный padding не раздвигает соседние строки, а налазит/запрыгивает на/под соседние строки.
margin по горизонтали раздвигает элементы, а по вертикали не применим.

  • <span></span>
    не имеет стандартной стилистики
    блок без смыслового значения служит только для обёртки.

    em - как измерение для шрифтов считается от родительского шрифта. Если исспользуется для отступа то берется половина от текущего шрифта. rem - как em только берется от стандартного html во всех случаях.

inline-block elements

имеет строчные отступы
Синтетический тип элементов. Занимают минимально необходимое место для контента.
Можно выравнивать по вертикали с помощью свойства vertical-align.
Разрешено задавать ширину и высоту.
line-height меняет высоту элемента и как следствие раздвигает соседние элементы.
padding увеличивает обыем элемента и как следствие раздвигает соседние элементы.
margin раздвигает элементы

Rules

  • Если переопределить блочный елемент инлайновым то не работает высота и ширина.
