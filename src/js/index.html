Concepts and rules

  • объявление переменных

    var - использовался раньше. Есть неявные проблемы с перезаписыванием.
    let - объявление переменных которые в процесе работы скрипта будет менятся
    const - объявление константы т.е. переменной которая после присвоения ей значения не позволяет изменять её значение

  • types of data

    • Number

      Infinity число которое не вмещается в 64bit, разное для браузеров.
      NaN - не число. Неявный тип данных.

      NaN назначается при ошибке математических вычислений. Т.е. делении на 0, на строку с буквами или со спецсимволами, единственная не равна сама себе.
      для проверки на NaN используй функцию isNaN(num);
      для проверки на NaN и на Infinity используй функцию isFinite(num);
      Если мы хотим округлить на n знаков после запятой то нужно это число умножить на число с n нолями,
      округлить и разделить на число с n нолями. Или исспользовать функцию num.toFixed(n) и привести результат из строки обратно к числу.

      Math.random() - генерирует случайные числа от 0 < 1.
      Если нужно изменить диапазон то * n расширит верхнюю границу диапазона до числа n, а + n увеличит обе границы на n.

        numeral literals

        4.2e3 or 4.2e-3 or .42e4 or 0.42e4 = 4200

      • decimal.toString(n) - переводит десятичное число в число системы исчисления основа которой указана в n.

      • Бинарная

        0b101010 = 42

      • Восьмиричная

        0o52 = 42

      • Десятичная

        42

      • Хекс или шеснадцатиричная

        0x2a or 0x2A = 42

    • String

      Для проверки на пустую строку и на строку с пробелами можно использовать str.trim();

  • rules

    в зависимости от типа формально записываемые значения в переменные называются числовым и строковым литералом

  • в зависимости от типа формально записываемые значения в переменные называются числовым и строковым литералом

  • оператор if в условии проверяет выражение по принципу true/false

  • null имеет тип данных object.

  • склеивание троки с строкой(числом) называется конкатенация

  • объявлять функции можно 3мя способами:

    - function name(...vars) {} - function declaration
    - const name = function(...vars) {}; - function expression
    - const name = (...vars) => {}; - function expression
    expression можно использовать только после объявления в отличии от declaration

  • функция выбора

    if () {
    } else if () {
    } else {}
    условный оператор выбора: условие ? true : false; другое название тернарный оператор.

  • Циклы бывают:

    - for (let i; i < n; i++) {} - можно упустить условия но не забудь поставить ';'.
    - for (const el of/is array/obj) {} - цикл без использования индекса.
    - while () {} or do {} while () - цикл неопределенного кол. итераций с пред/после условием
    переменные объявленные в цикле вне цикла не существуют

Commands

  • debugger; - запускает скрипт пошагово.

  • 'use strict'; - директива включает извещение об ошибках.

  • document.write('text'); - input text to screen. Используется редко потому что контент попадает на страницу иным образом.

  • console.log('text'); - input text to console.
    При перечислении переменных после запятой в кобках команда отображает их значения с пробелом от предыдущего

  • typeof(var); or typeof var; - возвращает строку с названием типа переменной. Работает с выражениями.

string

  • text.charAt(index); - возвращает символ строки по его индексу. Если за длинной строки возвращает "". text[index] - возвращает символ строки по его индексу. Если за длинной строки возвращает undefined.

  • text.substr(startIndex, length); - возвращает подстроку длинной length.
    text.substring(startIndex, endIndex); - возвращает подстроку в рамках строго меньше endIndex.
    text.slice(startIndex, endIndex); - возвращает подстроку в рамках строго меньше endIndex.
    Может работать без параметров и возвращать всю строку. slice позволяет работать с отрицательными аргументами.

  • text.length - возвращает длинну строки.

  • text.toLowerCase(); - преобразует текст в нижний регистр
    text.toUpperCase(); - преобразует текст в верхний регистр
    text.toLowerCase() !== text.toUpperCase() - проверка что символ не буква

  • В строке переприсваивать значения символов запрещено. Делается через специальные функции или при проборе в цикле.

    • text.include(part, index) - возвращает булево true если последовательность символов есть в любой части строки.
      Начинается проверка с индекса. Параметр индекс не обязателен.

    • text.indexOf(part, index) - возвращает индекс первого найденого совпадения подстроки. Начиная с индекса.
      text.lastIndexOf(part, index) - возвращает индекс последнего найденого совпадения подстроки. Начиная с индекса.
      Если нет совпадения возвращает -1.

    • text.startWith(part) - возвращает true если строка начинается с подстроки.
      text.endsWith(part) - возвращает true если строка заканчивается подстрокой.

    • text.trim() - убирает вначале и вконце строки пробелы
